Agenda de contatos Javascript

Autor: Leonardo Correa

Fiz essa agenda para O Projeto 4 da matéria de Laboratório de Computação 2. Com o framework Electron, foi possível criar uma aplicação para windows também o instalador com as linguagens Html5, CSS e javascript.
Eu desenvolvi normalmente o código como se fosse uma página web normal: fiz a interface frontEnd com html e css, e depois para fazer a lógica do sistema de agenda inseri na página um arquivo .js. Depois, utilizei o Electron para gerar a aplicação Windows.


Empresa de Aviação

Autor: Gustavo Roque

Projeto que simula uma empresa de aviação
Onde se pode adicionar aeronaves e verificar seus estados como:
- Adicionar ou remover aeronaves
- verificar se estão em voo ou estacinados
- pode embarcar, desembarcar e definir a quantidade maxima de passageiros
- adicionar em uma frota de aeronaves

Agencia Bancária

Autor: Gustavo Roque

Projeto que simula uma Agencia Bancaria. Onde se pode criar uma conta bancaria com as seguintes operações:
- Sacar e depositar na conta
- Consultar saldo
- Adicinar contas a agencia
- excluir contas da agencia
- Listar as contas cadastradas nessa agencia

Site com Bootstrap Studio

Autor: Leonardo Correa

Site que armei com o programa bootstrap Studio. Essa ferramenta cria toda a parte front end do site, e também tem a opção de adicionar Javascript. Tudo é feito com base no bootstrap, e também é possível ver e baixar códigos de outras pessoas online e colocar no seu projeto. Fica bem fácil criar as iplementações normais do bootstrap, como navbar e cards, pois o programa funciona com a lógica de arrastar, então você pode criar uma página toda apenas arrastando os elementos do bootstrap. O site pode ser acessado em: telecom.connsecurity.com
