Utilizando comentários no blog com Discus

O Discus é uma secção de coentários completamente funcional e com suporte a comentários via Facebook e Twitter, entre outros. Para adicionar isso ao blog, é bem simples.

Criar conta Discus

Para começar, é necessário cirar uma conta no site do Discus, onde, após criada, você irá receber um shortaname, que será usado depois.

Adicionar bloco de código ao blog

Primeiramente, precisamos do seguinte no nosso arquivo de configuração principal _condif.yml: 

url: http://blogpi2.site
root: /
disqus_enabled: true
disqus_shortname: nossoshortname

Note que é necessário colocar o shortname em uma das opções, pois é assim que o Discus irá associar os comentários com o site para a conta que fizemos.

Depois disso, basta Adicionar esse código onde fica a secção que representa aonde você irá querer os comentários: 

<% if (!index && post.comments && config.disqus_enabled && config.disqus_shortname){ %>
<section id="comments">

<div id="disqus_thread"></div>
<script>

var disqus_config = function () {
    this.page.url = '<%= config.url + '/' + post.canonical_path %>';
    this.page.identifier = '<%= post.id || post.canonical_path %>';
    this.page.title = '<%= post.title %>';
};

(function() {
var d = document, s = d.createElement('script');
s.src = 'https://<%= config.disqus_shortname %>.disqus.com/embed.js';
s.setAttribute('data-timestamp', +new Date());
(d.head || d.body).appendChild(s);
})();
</script>
<noscript>Please enable JavaScript to view the <a href="https://disqus.com/?ref_noscript">comments powered by Disqus.</a></noscript>
</section>
<% } %>

Depois disso, o Hexo irá gerar a secção de comentários no fim de cada Blog.
